-------------------------------------------------------------------------------
-- About
-------------------------------------------------------------------------------
TwentyEleven is a Drupal theme based on Wordpress' TwentyEleven theme.


-------------------------------------------------------------------------------
-- TwentyEleven features
-------------------------------------------------------------------------------
Besides the default Drupal theme features, the following additional features 
are available.
* Front page has a choice of One column/ Two column layout
* Custom header images
* Drop down menus
